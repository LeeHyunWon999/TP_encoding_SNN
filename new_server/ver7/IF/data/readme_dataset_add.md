새 데이터셋 추가 시 필요한 작업들
- 모델에 들어갈 수 있도록 미리 전처리
- data_loader.py : 데이터로더 작성 (이미 정제된 데이터인 경우 여기서 추가 변형 작업 수행 가능 : 0~1 일반화 등)
- util.py : get_data_loader_train(혹은 test) 에 해당 데이터로더 분기 동작 추가하기
- model.py : 해당 데이터셋이 새로운 형태의 모델을 요구하는 경우(차원 매칭 등), 여기에 변형 버전 새로 만들기
  - util.py : 모델이 추가되는 경우 get_model 에도 분기 추가하고, propagation에서도 모델 타입 따라 순전파 방법 고안할 것 (웬만하면 1차원+시간, 2차원+시간 정도로 끝날듯 하지만 필요한 경우 추가할 것)

- config : 각 데이터셋과 모델에 맞는 세팅 및 폴더 각각 만들어서 재지정하기(데이터로더, 모델 등에 따라 많이 달라질 수 있으니 기존 config들 참고하며 작업할 것!)
